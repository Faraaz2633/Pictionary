Who are we

Pictionary is a simple picture based dictionary. It consists of images belonging to different catagories, along with valid information about them. Yes, it is an open source project! We are trying to keep it informational and educational as much as possible. It is an awesome infographic encyclopedia made from your contributions for you!

Tech Stack

Keeping things clean, simple and beginner friendly we have decided to start with the following tech stack. Surely other necessary technologies would be added if needed.
Tech Stack: HTML5, CSS3, JavaScript, Bootstrap

Hacktoberfest

This project has been a part of Hacktoberfest 2021. Hacktoberfest is a month long virtual festival event to celebrate open source contributions presented by Digital Ocean, Intel and DEV. It is the easiest way to get into open source!

JWoC

This project has been a part of JWoC 2022. JWoC (JGEC Winter of Code) is a coding event organised annually by Jalpaiguri Government Engineering College which helps students to plunge into Open Source contribution and get the realm of Software Development.
